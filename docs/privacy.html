Word Counter — Privacy Policy

Last updated: June 28, 2026

Word Counter is a Chrome extension that counts words, characters, sentences, and paragraphs from text you select or paste. All text analysis happens locally in your browser. We do not collect, store, or transmit your text.

Data We Collect

None. The extension does not collect personally identifiable information, usage analytics, or the content of pages you visit.

Data Stored on Your Device

The extension saves only your preferences using Chrome's local storage (chrome.storage.local):

  • Reading speed (words per minute)
  • Speaking speed (words per minute)
  • Custom character limit

Text you select, paste, or type is processed in memory and is never saved to disk or synced anywhere.

Data Transmitted

None. The extension does not connect to external servers, APIs, or third-party analytics services.

Website Access

The extension injects a lightweight content script on web pages so it can read text you explicitly select — for example when you open the popup, use the keyboard shortcut (Ctrl+Shift+W / Cmd+Shift+W), or choose "Count Selected Words" from the right-click menu.

It does not read your full page content, browsing history, or keystrokes. Access occurs only when you initiate a count action.

Permissions

  • storage — Save your reading/speaking speed and custom limit settings.
  • contextMenus — Add a "Count Selected Words" option to the right-click menu.
  • activeTab — Read the currently selected text on the active tab when you request a count.
  • notifications — Show count results as a system notification when an on-page tooltip cannot be displayed.

Children's Privacy

This extension is not directed at children under 13 and does not knowingly collect any information from anyone.

Changes to This Policy

We may update this policy if the extension changes. The "Last updated" date at the top will reflect any revision. Continued use after changes constitutes acceptance of the updated policy.

Voluntary Support

The extension may link to an external Ko-fi page for optional tips. That site has its own privacy policy; Word Counter does not collect or process any payment information.
